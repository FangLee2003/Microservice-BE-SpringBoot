1. IntelliJ IDEA
- Choose File > Project Structure > Java SDK 17
- Choose Current File > Edit Configurations (Run/Debug Configurations) > Add New Configuration > Maven
- Right click on project folder:
+ Choose Build Model ...
+ Choose Maven > Generate Sources and Update Folders
2. Axon Server
- Download the axon server zip file at https://developer.axoniq.io/download
- Extract to any location and open cmd there
- Type java -jar axonserver.jar
